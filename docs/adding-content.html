How to Add a New Audio Project

Content Manager Guide

Overview

All audio content is managed through the Content Manager — a password-protected admin tool built directly into the site.

No code editing required.

1

Access the Content Manager

  1. Visit consentingadults.netlify.app
  2. Click Tools in the top navigation bar
  3. Enter the admin password when prompted
  4. You are now inside the Content Manager
2

Open the Add Form

Click the "+ Add New Entry" button near the top of the Content Manager page.

3

Fill in the Details

Title (required)

The name of the audio piece.

Example: Late Night Confession

Artist (required)

Select the creator from the dropdown. This determines which artist profile the audio appears under.

Short Description (max 100 characters)

A punchy line displayed in the catalog card. Think of it as your hook.

Example:
She had one confession. You had no idea what was coming.

Long Description

The full description shown on the detail page. Include mood, setup, content notes, or anything that helps listeners decide.

Tags

Tags power the catalog filtering system.

  • Gender/audience tag always comes first — enforced automatically
  • Type a tag and press Enter or Comma to add it
  • Existing tags are suggested as you type — reuse them to avoid duplicates
  • Aliases are normalized automatically (e.g. gfe → GFE, rp → roleplay)

4

Add Platform Links

Click "+ Add Platform" for every location where the audio is hosted.

Field What to Enter
Platform Select from the dropdown (Reddit, SoundCloud, Patreon, etc.)
URL Paste the direct link to that audio on the selected platform

Add as many platform entries as needed. The catalog will display all available links.

5

Save

Click Save Entry.

The audio will immediately appear in:

  • The main catalog (/catalog.html)
  • The selected artist's profile page under Audio Works

Editing an Existing Entry

  1. Find the audio entry in the Content Manager list
  2. Click Edit
  3. Make your changes
  4. Click Save Entry

Tips

Duplicate Tags Check the Existing Tags row before creating a new tag. Reuse existing tags whenever possible.
Short Description Write this after finishing the long description. It's much easier to distill once you have the full version written.
Platform Links One platform link is perfectly acceptable. Add more as the audio is published elsewhere.
Artist Selection Verify the correct artist is selected. Entries only appear on that artist's profile page.

Who Has Access?
