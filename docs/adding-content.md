# How to Add a New Audio Project to Consenting Adults

## Overview

All audio content is managed through the **Content Manager** — a password-protected admin tool built into the site. No code editing required.

---

## Step 1 — Access the Content Manager

1. Go to [consentingadults.netlify.app](https://consentingadults.netlify.app)
2. Click **Tools** in the top navigation bar
3. Enter the admin password when prompted
4. You're now in the Content Manager

---

## Step 2 — Open the Add Form

Click the **"+ Add New Entry"** button near the top of the Content Manager page.

---

## Step 3 — Fill in the Details

### Title *(required)*
The name of the audio piece.
> Example: `Late Night Confession`

### Artist *(required)*
Select the creator from the dropdown. This determines which artist profile the audio appears under.

### Short Description *(max 100 characters)*
A single punchy line shown in the catalog card. Keep it tight — think of it as a hook.
> Example: `She had one confession. You had no idea what was coming.`

### Long Description
The full description shown on the detail view. Can include mood, setup, content notes — whatever helps the listener decide.

### Tags
Tags drive the filter system in the catalog. Follow these rules:

- **Gender/audience tag always goes first** — the form enforces this automatically
  - Common values: `M4F`, `F4M`, `M4M`, `F4F`, `Nonbinary`
- Type a tag and press **Enter** or **comma** to add it
- The form will suggest existing tags as you type — use those to avoid duplicates
- Common tags: `ASMR`, `GFE`, `roleplay`, `comfort`, `soft spoken`, `L-bombs`, `SFW`, `NSFW`
- Aliases are normalized automatically (e.g. `gfe` → `GFE`, `rp` → `roleplay`)

> Example tag set: `F4M`, `GFE`, `ASMR`, `comfort`, `soft spoken`

---

## Step 4 — Add Platform Links

Click **"+ Add Platform"** for each place the audio is hosted.

| Field | What to enter |
|-------|---------------|
| Platform | Select from the dropdown (Reddit, SoundCloud, Patreon, etc.) |
| URL | Paste the direct link to that audio on that platform |

You can add as many platforms as needed. The catalog will display all of them.

---

## Step 5 — Save

Click **Save Entry**. The audio will appear immediately in:
- The main **catalog** (`/catalog.html`)
- The artist's **profile page** under their Audio Works section

---

## Editing an Existing Entry

1. In the Content Manager, find the entry in the list
2. Click **Edit**
3. Make your changes
4. Click **Save Entry**

---

## Tips

- **Duplicate tags** — always check the "Existing tags" chip row before adding a new one. If it's already there, click it to reuse it rather than typing a new variation.
- **Short description** — write this last, after the long description. It's easier to distill once you've written the full version.
- **Platform links** — if an audio is only on one platform, one link is fine. Add more as you post to additional platforms.
- **Artist selection** — make sure you pick the correct artist. The entry will only appear on that artist's profile page.

---

The Content Manager is password-protected. Contact the site admin (JOI Electric) for the password.
